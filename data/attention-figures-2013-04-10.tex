Mean squared error averaged over all testing data versus (constant) precision (in bits) of the network. Error is on a log scale. For a given set size $s$, network is trained on $2^{s-3}$ of the $2^s$ possible inputs (selected at random); tested on the remaining inputs. Learning rate $\eta=4.0$, 400 iterations over training data. The first bit of each input represents the distance (bit in the string) that should be attended to (all 0 in these experiments). Hidden layer contains 20 neurons.
